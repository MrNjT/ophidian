\subsection{Detailed Description}
This class provides the basic interface for manipulating a 3\-D routing grid including edges capacities and demands. It handles layers in horizontal and vertical directions. This version assumes that even layers are routed horizontally while odd layers are routed vertically. Layers are numbered from 0 to n-\/1. For a 2\-D routing, the user can use an unique layers and merge all the capacities in that layer.
